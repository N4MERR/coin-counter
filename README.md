# Coin Identifier

A desktop application that uses a pre-trained AI model to identify coins from images. Built with Python and PySide, the app allows users to upload one or multiple images and quickly determine what coin they are looking at.

---

## Features

* **AI-Based Coin Identification**
  Uses a pre-trained machine learning model to recognize coins from images.

* **Single and Batch Image Upload**
  Upload one image or multiple images at once for bulk identification.

* **Fast Processing**
  Returns predictions in seconds.

* **User-Friendly Interface**
  Simple desktop interface built with PySide.

* **Prediction Confidence**
  Displays confidence scores for each identified coin.

---

## How It Works

1. **Upload Images**
   Select one or more images of coins.

2. **Image Processing**
   The app preprocesses each image to extract relevant features.

3. **Prediction**
   The AI model classifies each coin and determines its identity.

4. **Results**
   The application displays the predicted coin type along with confidence scores.

---

## Tech Stack

* **Language:** Python
* **UI Framework:** PySide
* **AI Model:** YOLOv8
* **Image Processing:** OpenCV

---

## Usage

1. Launch the application.
2. Upload one or multiple images of coins.
3. View the identified coin types and confidence scores.

---

## Example

**Input:**

* Images of Czech coins

**Output:**

* Image 1: 10 Czech Koruna (Confidence: 94%)
* Image 2: 20 Czech Koruna (Confidence: 91%)

---

## Notes

* Use clear, high-quality images for best results.
* Ensure the coin is fully visible and centered.
* Avoid shadows, glare, and reflections.

---

## Disclaimer

Predictions are generated by an AI model and may not be 100% accurate. Results should be considered as guidance only and verified when necessary.
